# 장바구니 / 주문 / 결제 기능 명세서

---

## 장바구니 기능

| 기능 ID | 기능명 | 설명 | 입력 | 출력 | 주요 처리 로직 | 예외 처리 |
|---|---|---|---|---|---|---|
| CART-01 | 장바구니 담기 | 상품을 장바구니에 추가 | userId, productId, optionId, quantity | cartItemId | 동일 상품+옵션 존재 시 수량 증가, 없으면 신규 생성 | 상품 없음, 옵션 없음, 수량 < 1 |
| CART-02 | 장바구니 조회 | 장바구니 목록 조회 | userId | cartItem 리스트 | 상품 정보, 옵션, 수량, 가격 반환 | 장바구니 없음 |
| CART-03 | 수량 변경 | 상품 수량 수정 | cartItemId, quantity | 수정된 cartItem | 수량 1 이상 유지 | 수량 < 1, 상품 없음 |
| CART-04 | 단일 삭제 | 특정 상품 삭제 | cartItemId | 삭제 성공 여부 | 해당 항목 삭제 | 항목 없음 |
| CART-05 | 선택 삭제 | 선택 상품 삭제 | cartItemIds | 삭제 성공 여부 | 여러 항목 삭제 | 일부 항목 없음 |
| CART-06 | 전체 삭제 | 장바구니 전체 삭제 | userId | 삭제 성공 여부 | 전체 항목 삭제 | 없음 |
| CART-07 | 선택 주문 | 선택 상품 주문 | cartItemIds | 주문 요청 데이터 | 선택 상품만 주문으로 전달 | 선택 상품 없음 |
| CART-08 | 전체 주문 | 전체 상품 주문 | userId | 주문 요청 데이터 | 전체 상품 주문으로 전달 | 장바구니 비어있음 |
| CART-09 | 가격 계산 | 장바구니 금액 계산 | cartItemIds | totalPrice | 상품 가격 × 수량 합산 | 없음 |
| CART-10 | 상품 상태 반영 | 상품 상태 표시 | productId | 상태 정보 | 품절/삭제 여부 반영 | 없음 |

## 주문 기능

| 기능 ID | 기능명 | 설명 | 입력 | 출력 | 주요 처리 로직 | 예외 처리 |
|---|---|---|---|---|---|---|
| ORDER-01 | 주문 생성 | 장바구니 기반 주문 생성 | userId, cartItemIds | orderId | 주문 생성, 상태 PENDING 설정 | 상품 없음, 권한 없음 |
| ORDER-02 | 주문 상품 생성 | 주문 상품 구성 | orderId, 상품 정보 | orderItem 리스트 | 각 상품을 OrderItem으로 저장 | 없음 |
| ORDER-03 | 상품 스냅샷 저장 | 주문 당시 상품 정보 저장 | product 정보 | 저장 결과 | 이름, 가격, 옵션, 이미지 저장 | 없음 |
| ORDER-04 | 상품 검증 | 상품 유효성 검사 | productId | 검증 결과 | 상품 존재 여부, 판매 상태 확인 | 상품 없음, 판매 중지 |
| ORDER-05 | 가격 검증 | 최신 가격 반영 | productId | 최신 가격 | DB 기준 가격 적용 | 없음 |
| ORDER-06 | 주문 금액 계산 | 총 주문 금액 계산 | orderItem 리스트 | totalAmount | 가격 × 수량 합산 | 없음 |
| ORDER-07 | 주문 목록 조회 | 사용자 주문 조회 | userId | order 리스트 | 주문 목록 반환 | 없음 |
| ORDER-08 | 주문 상세 조회 | 주문 상세 조회 | orderId | order 상세 | 주문 + 상품 정보 반환 | 주문 없음 |

## 결제 기능 (토스)

| 기능 ID | 기능명 | 설명 | 입력 | 출력 | 주요 처리 로직 | 예외 처리 |
|---|---|---|---|---|---|---|
| PAY-01 | 결제 준비 | 결제 요청 데이터 생성 | orderId | 결제 요청 데이터 | amount, orderId 생성 | 주문 없음 |
| PAY-02 | 결제 요청 | 토스 결제창 호출 | orderId, amount | 결제창 | 결제창 redirect | 금액 오류 |
| PAY-03 | 결제 성공 처리 | successUrl 처리 | paymentKey, orderId, amount | 처리 결과 | orderId/amount 검증 | 금액 불일치 |
| PAY-04 | 결제 승인 | 결제 승인 API 호출 | paymentKey, orderId, amount | 승인 결과 | 토스 confirm API 호출 | 승인 실패 |
| PAY-05 | 결제 완료 처리 | 결제 성공 반영 | orderId | 성공 상태 | Order: PAID 변경 | 없음 |
| PAY-06 | 결제 실패 처리 | failUrl 처리 | errorCode | 실패 상태 | Payment FAILED 처리 | 없음 |
| PAY-07 | 결제 상태 관리 | 상태 변경 관리 | paymentId | 상태 값 | READY→DONE/FAILED | 없음 |
| PAY-08 | 중복 결제 방지 | 중복 결제 차단 | orderId | 처리 결과 | 이미 PAID면 차단 | 중복 요청 |
| PAY-09 | 금액 검증 | 금액 무결성 확인 | amount | 검증 결과 | 서버 금액과 비교 | 불일치 |
| PAY-10 | 장바구니 정리 | 결제 후 장바구니 삭제 | cartItemIds | 삭제 결과 | 결제 성공 시 삭제 | 없음 |

## 전체 기능 구조

| 구분 | 기능 |
|---|---|
| 장바구니 | 담기, 조회, 수정, 삭제, 선택 주문 |
| 주문 | 생성, 검증, 스냅샷, 금액 계산, 조회 |
| 결제 | 요청, 승인, 성공/실패 처리, 검증, 중복 방지 |
