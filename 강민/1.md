# 강민 - 무신사 1만건 크롤링 및 최종 CSV 전처리

## 구현한 기능

각자 담당 쇼핑몰 기준으로 무신사 상품 데이터를 크롤링하고, 병합 가능한 공통 CSV 스키마로 전처리한 뒤 DB 조회 성능 측정까지 진행했다.

수집 데이터 활용 목적:

1. AI 유사도 기반 상품 추천 기능 개발
2. 성능 테스트
3. `source_url` 기반 상세 재크롤링

## CSV 스키마

| 컬럼 | 타입 | 필수 여부 | 설명 |
| --- | --- | --- | --- |
| product_name | String | 필수 | 상품명 |
| brand | String | 필수 | 브랜드명 |
| category | String | 필수 | 대카테고리 |
| sub_category | String | 선택 | 소카테고리 |
| price | Int | 필수 | 가격, 숫자만 저장 |
| discount_rate | Float | 선택 | 할인율 |
| rating | Float | 선택 | 평점 |
| review_count | Int | 선택 | 리뷰수 |
| tags | String | 선택 | 태그 |
| description | String | 선택 | 상품 설명 |
| image_url | String | 선택 | 이미지 URL |
| source | String | 필수 | 출처 쇼핑몰 |
| source_url | String | 필수 | 상품 원본 URL |

선택 컬럼 수집 불가 시 `None` 문자열로 통일했다.

## 적용한 크롤링/전처리 기준

| 항목 | 적용 내용 |
| --- | --- |
| 수집량 | 스크립트 기본 목표 10000건 |
| 품절 상품 | `isOutOfStock` 기준 제외 |
| 중복 상품 | 같은 브랜드 + 같은 상품명 기준 1개만 저장 |
| 인코딩 | UTF-8-sig |
| 날짜 포맷 | 결과 리포트 날짜는 YYYY-MM-DD |
| 가격 | 콤마/원 제거 후 숫자만 저장 |
| null 처리 | 빈 문자열 없이 `None`으로 통일 |
| source | `musinsa` 고정 |
| 카테고리 | `category`, `sub_category`는 무신사 원본 그대로 저장 |

## 실행 명령

```powershell
python 강민\crawling\collect_sitemap_urls.py --limit 10000 --output 강민\data\raw\musinsa_product_urls.txt
python 강민\crawling\musinsa_crawler.py --url-file 강민\data\raw\musinsa_product_urls.txt --output 강민\data\raw\musinsa_products_raw.csv --delay 1.0 --retries 1 --target-count 10000 --resume
python 강민\merge_clean\clean_products.py --input 강민\data\raw\musinsa_products_raw.csv --output 강민\data\processed\musinsa_products_clean.csv
python 강민\db\import_products.py --csv 강민\data\processed\musinsa_products_clean.csv --db 강민\data\db\musinsa.sqlite3
python 강민\db\import_products.py --csv 강민\data\processed\musinsa_products_clean.csv --db 강민\data\db\musinsa_no_index.sqlite3 --no-indexes
python 강민\db\benchmark_queries.py --db 강민\data\db\musinsa.sqlite3 --iterations 1000 --output 강민\data\reports\query_benchmark_indexed.csv --compare-db 강민\data\db\musinsa_no_index.sqlite3 --compare-output 강민\data\reports\query_benchmark_no_index.csv --summary-output 강민\data\reports\db_performance_result.md
```
